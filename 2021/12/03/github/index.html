更新Github上的代码

一、克隆自己Github上的项目代码到本地

原因嘛，Github上不能删除文件夹的，只能把项目拉到本地删除再更新

  1. 首先随便创建一个文件夹(存放下载下来的代码)

  2. 输入指令克隆自己的代码

    git clone https://github.com/用户名/仓库名.git

二、查看一下仓库状态

git status

注意要先切换到存储原来GitHub本地代码的位置，不是刚刚下载代码才建的文件夹。

三、更新

  1. 直接在本地，把改好的文件替换原来的相应部分就好

    git add *--表示更新全部

  2. 添加更新说明，commit只是将其提交到缓存区域

    git commit -m "update"(可将update替换为别的内容)

  3. 上传

    git push origin 分支名 --(若是多人同时开发维护，首先git pull，拉取当前分支最新代码)
